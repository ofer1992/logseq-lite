# Logseq Lite

A lightweight, fast web viewer for your Logseq notes. Built with Python and FastHTML, this application provides a clean and simple interface to browse your Logseq knowledge base.

## Features

- 🚀 Fast and lightweight web interface
- 🔗 Wiki-style link support ([[links]])
- 📅 Journal entries viewer
- 🔍 Search functionality
- 🌓 Light/dark mode support

## Usage

```bash
python app.py <path-to-logseq-folder>
```

Replace `<path-to-logseq-folder>` with the path to your Logseq notes directory.

## Requirements

- Python 3.x
- FastHTML
